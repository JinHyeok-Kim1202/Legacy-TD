[조합 규칙 확정본]
1. 모든 조합은 "선택한 유닛 1개 + 나머지 재료" 기준으로 본다.
2. 재료 범위는 storage + board 전체 기준이다.
3. 같은 재료가 storage와 board에 동시에 있으면 storage부터 먼저 소모한다.
4. 결과 유닛은 선택한 유닛의 위치에 생성한다.
   - 선택 유닛이 storage에 있으면 같은 storage 위치
   - 선택 유닛이 board에 있으면 같은 board 좌표
5. 재료 개수 규칙
   - 레어: 커먼 3개
   - 유니크: 레어 3개
   - 레전더리: 유니크 4개
   - 초월자 / 불멸자 / 신 / 해방자: 레전더리 3개

[밸런스 메모]
1. 커먼 단계는 책/남자에 과도하게 몰리던 비율을 줄이고,
   아이 / 노인 / 새 / 불의 사용량을 올려 초반 재료 분포를 더 고르게 맞췄다.
2. 유니크 단계는 같은 등급 재료가 섞이던 문제를 없애고,
   모든 조합이 레어 3개만 사용하도록 정리했다.
3. 레전더리 단계는 철학자 / 왕 / 지도자 쏠림이 여전히 있지만,
   이전보다 운동선수 / 가수 / 마법사 / 전략가 / 연구원 축도 함께 올라오도록 분산했다.
4. 상위 4등급은 잔다르크 / 손자 쏠림이 완전히 사라지진 않지만,
   레오나르도 다빈치 / 프레디 머큐리 / 마이클 조던 / 니콜라 테슬라 같은
   이전 저활용 레전더리도 더 자주 쓰이도록 재배치했다.
